1. Content 관련 메소드

1) html() 메소드

선택된 요소의 content 영역(시작 태그 ~ 종료 태그 사이)을 리턴하거나 변경하는 메소드이다.
html 태그까지 포함해서 읽어오거나, 변경할 문구에 html 태그가 존재하려면 태그로 인식해서 설정한다.

2) text() 메소드

선택된 요소의 content 영역(시작 태그 ~ 종료 태그 사이)을 리턴하거나 변경하는 메소드이다.
html 태그는 무시하고 텍스트만 읽어오거나, 변경할 문구에 html 태그가 있으면 태그도 문자열로 인식해서 설정한다.
