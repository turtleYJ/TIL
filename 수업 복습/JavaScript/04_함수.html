화살표 함수(Arrow Function)

        ES6에 도입된 선언 방법으로 function 키워드 대신 => 를 사용해 좀 더 간략한 방법으로 함수를 선언할 수 있다.

        [표현법] 
            - 매개 변수가 없을 때 : () => { ... } 
            - 매개 변수가 하나 일 때 : x => { ... }
            - 매개 변수가 두 개 이상일 때 : (x, y) => { ... }
            - 리턴 처리
                - 처리할 라인이 여러 개라면 {}로 감싸서 처리해야 한다.
                (x, y) => {
                     실행구문;
                     
                     return x*y;
                }
                - 처리할 라인이 하나라면 {}와 return 문을 생략할 수 있다.
                (x, y) => x * y;

        * 기준 function과 this가 가리키는 대상이 조금 다르기 때문에 주의해야 한다. (검색. javascript arrow function this)
